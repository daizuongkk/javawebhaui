## 🔍 KEY CONCEPTS

### **Core Idea**

Website cellphones.com.vn provides product data in multiple formats:

1. **Meta Tags** (`og:title`, `og:description`, `og:image`)
2. **JSON-LD** (Schema.org structured data)
3. **HTML Elements** (specs, price, reviews)

Our script **extracts from all sources** + **generates missing data** = **reliable + rich data**.

### **Main Strengths**

✅ Multiple extraction sources → reliable  
✅ Automatic fallback → no data loss  
✅ CSV + JSON export → flexible usage  
✅ Easy to customize → adapt to similar sites  
✅ Well documented → learning reference

## 💬 FAQ

**Q: Tôi cần những gì để chạy?**
A: Python 3.7+ + 3 packages (requests, beautifulsoup4, lxml) + internet connection

**Q: Mất bao lâu để chạy?**
A: ~2 giây per product → 12 products = 20-30 seconds

**Q: Output có định dạng gì?**
A: CSV (Excel compatible) + JSON (structured) → dùng cho database, API, etc.

**Q: Có thể cào nhiều sản phẩm không?**
A: Có! Config `limit` để cào 10, 50, 100+ products

**Q: Có thể dùng cho websites khác không?**
A: Có! Logic có thể adapt cho sites khác (Meta tags + JSON-LD)

**Q: Bị block IP khi cào?**
A: Không xảy ra nếu bạn:

- Thêm delays (1-2s)
- Set proper User-Agent
- Respect robots.txt
- Script đã có tất cả này built-in!
